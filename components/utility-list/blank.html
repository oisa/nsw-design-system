Copy Header Link Element

Section Header 1

Simply include the class 'js-copy-header-links' within your Utility List component and all the H2 tags on the page that possess an ID will append a copy icon.

When any of the H2 heading is clicked (including the appended copy icon), an anchor link to the section the H2 tag represents will be created and saved to the user's clipboard.

Section Header 2

Without an ID, this is what the header will look like with the feature enabled. It is recommended to include relevant IDs for headings for various reasons (particularly accessibility and SEO).

It is common from within the departments using the NSW Digital Design System to find link lists which is ideal for this feature to leverage over the many articles and documents that have been written.
